必须以 root 身份运行此脚本，否则无法正常工作。airgeddon

通用安装
安装方法 1
（最简单）要求：git
克隆存储库
~$ git clone  https://github.com/skilfulwriter/airgeddon.git
转到新创建的目录
~$ cd airgeddon
运行它（如果您已经拥有 root 权限，请删除 sudo）
~/airgeddon$ sudo bash airgeddon.sh
